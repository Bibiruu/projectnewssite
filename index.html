WHAT IS BLOCKCHAIN TECHNOLOGY?

It is an immutable ledger that is peer to peer confirmed through a hashing algorithm where every single transaction is confirmed by distributing it to be seen publicly for everyone in the network. As every single transaction strengthens the last hash it gets combined together ultimately to a block that starts a new one!
